Wiloquery

Bibliothèque JavaScript fournissant un objet JSON pour les paramètres de l'URL.

Télécharger Code source Exemple

Description

La bibliothèque JavaScript Wiloquery ajoute la propriété query à l'objet window.location. query contiendra une représentation JSON de la variable window.location.search. Pour chaque paramètre de l'URL, une propriété est ajoutée dans l'objet JSON. Si un paramètre est présent plusieurs fois dans l'URL : les valeurs sont regroupées dans un tableau. Pour les paramètres n'ayant pas de valeur : null est stocké dans l'objet.

Par exemple pour l'URL http://www.example.org/?a=x&b=y&b=z&c=&d&&, window.location.query contiendra : 


{
    "a": "x",
    "b": [ "y", "z" ],
    "c": "",
    "d": null
}

Installation

Il suffit de télécharger le script et de l'inclure dans la page (avant la première utilisation de la variable window.location.query). 


<!DOCTYPE html>
<html>
  <head>
    <!-- ... -->
    <script src="wiloquery.min.js"></script>
    <!-- ... -->
  </head>
  <body>
    <!-- ... -->
  </body>
</html>
