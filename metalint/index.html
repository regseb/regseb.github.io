Metalint

Un linter pour les gouverner tous.

Metalint est un outil pour vérifier tous les fichiers de votre projet.

Cinq langages peuvent être vérifiés grâce à 13 linters. Metalint est hébergé dans le gestionnaire de paquets npm. La configuration se fait avec des fichiers JSON regroupés dans un répertoire. Metalint s'utilise principalement en ligne de commande. Plusieurs formats de sortie sont disponibles : CSV, Checkstyle, JSON, ...
