Metalint

Metalint est un outil analysant tous les fichiers de votre projet. Selon le type de fichier (JavaScript, HTML, CSS…), des linters (outils d'analyse statique) externes sont utilisés pour contrôler le code source. Metalint agrège les résultats ; les erreurs sont ainsi regroupées par fichier et affichées dans le même format.

  • 10 langages / technologies peuvent être analysées grâce à 20 linters.
  • Une seule commande pour analyser tous les fichiers de votre projet.
  • Metalint est hébergé dans le gestionnaire de paquets npm.
  • La configuration se fait avec des fichiers JSON regroupés dans un répertoire.
  • Plusieurs formats de sortie sont disponibles : CSV, Checkstyle, JSON…

TL;DR

Si votre projet a un fichier package.json et qu'il possède déjà un ou plusieurs linters : exécutez les commandes suivantes pour installer Metalint, initialiser la configuration (en tentant de récupérer la configuration des linters déjà installés) et exécutez metalint. 


npm install metalint --save-dev
npx metalint init
npx metalint
