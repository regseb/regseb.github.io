Metalint

Un linter pour les gouverner tous.

Metalint est un outil analysant tous les fichiers de votre projet. Selon le type de fichier (JavaScript, HTML, CSS, ...), des linters (outils d'analyse statique) externes sont utilisés pour contrôler le code source. Metalint agrège les résultats et les erreurs sont ainsi regroupées par fichier et affichées dans le même format.

  • Six langages peuvent être analysés grâce à 13 linters.
  • Une seule commande pour analyser tous vos fichiers.
  • Metalint est hébergé dans le gestionnaire de paquets npm.
  • La configuration se fait avec des fichiers JSON regroupés dans un répertoire.
  • Plusieurs formats de sortie sont disponibles : CSV, Checkstyle, JSON, ...
