Formateurs · checkstyle

Description

Ce formateur retourne un document XML ayant la structure des rapports de l'outil Checkstyle.

Une option est disponible : indent, pour choisir la taille des indentations :

  • -1 (par défaut) : le fichier n'aura pas de retours à la ligne ;
  • 0 : le document aura des retours à ligne, mais aucune indentation ;
  • 1 : des retours à ligne et une indentation d'une espace ;
  • 2 : une indentation de deux espaces ;
  • etc.

Exemple


{
    "reporters": {
        "formatter": "checkstyle",
        "options": { "indent": 2 }
    }
}

Deux fichiers sont analysés : global.js qui n'a aucune erreur et script.js qui contient deux erreurs (une remontée par JSHint et l'autre par JSCS). 


<?xml version="1.0" encoding="UTF-8"?>
<checkstyle version="6.8">
  <file name="global.js">
  </file>
  <file name="script.js">
    <error line="255" column="36" severity="error" message="Don&apos;t make functions within a loop." source="jshint.W083" />
    <error line="278" column="3" severity="error" message="Comments cannot contain the following keywords: todo, fixme" source="jscs" />
  </file>
</checkstyle>
