Description

Avec ce formateur, un extrait du code source est affiché pour chaque alerte remontée.

Deux options sont disponisbles :

  • showZeroNotice (false par défaut) pour afficher les fichiers sans alertes.
  • showNoChecked (false par défaut) pour afficher les fichiers non-analysés.

Exemple


{
    "reporters": {
        "formatter": "console",
        "options": { "showZeroNotice": true, "showNoChecked": true }
    }
}
