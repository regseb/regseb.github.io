Formateurs · csv

Description

Ce formateur retourne les données au format CSV. La première ligne contient le nom des colonnes et chaque ligne suivante correspond à une alerte. Les colonnes sont :

  • file : le nom du fichier ;
  • line : le numéro de la ligne ;
  • column : le numéro de la colonne ;
  • message : le message décrivant le problème ;
  • linter : le linter ayant trouvé le problème ;
  • rule : la règle du linter qui n'est pas respectée.

Les colonnes sont séparées par une virgule et les lignes par les caractères CRLF (\r\n). Les valeurs des colonnes file, message et rule sont enrobées par des guillemets. Si le contenu d'une colonne contient un guillemet, celui-ci est doublé pour l'échapper.

Exemple


{
    "reporters": {
        "formatter": "csv"
    }
}

Voici un rapport où deux alertes sont affichées pour le même fichier.
