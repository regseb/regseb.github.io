Linters

Liste

Actuellement Metalint gère 20 linters permettant d'analyser les fichiers :

  • Add-on de Firefox (WebExtension) : Add-ons Linter ;
  • CoffeeScript : CoffeeScript ;
  • CSS (SCSS, Sass, Less) : CSSLint, doiuse, lesshint, Purgecss et stylelint ;
  • HTML : HTMLHint et htmllint ;
  • JavaScript : ESLint, Flow, JSCS, JSHint et JavaScript Standard Style ;
  • JSON : JSON-Lint et JSONLint ;
  • Markdown : MarkdownLint ;
  • package.json (npm) : David DM ;
  • TypeScript : TSLint ;
  • YAML : YAML Lint.

NB : Les linters ne sont pas inclus par défaut dans Metalint. Vous devez installer ceux que vous allez utiliser. Consultez le détail de chaque linter pour savoir comment l'installer et le configurer.
