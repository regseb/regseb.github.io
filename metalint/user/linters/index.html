Linters

Introduction

Actuellement Metalint gère 13 linters permettant de vérifier les langages :

  • CSS : CSSLint et stytelint;
  • HTML : html5-lint, HTMLHint et htmllint ;
  • JavaScript : ESLint, Flow, JSCS, JSHint et JavaScript Standard Style;
  • JSON : JSON-Lint et JSONLint ;
  • Markdown : MarkdownLint.
