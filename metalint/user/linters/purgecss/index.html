Linters · Purgecss

Description

Purgecss est un outil pour détecter les règles CSS qui ne sont pas utilisées.

Autres linters CSS (SCSS, Sass, Less) : CSSLint, doiuse, lesshint et stylelint.

Installation

La commande suivante installe et ajoute Purgecss dans les dépendances de développement de votre projet npm. 


npm install purgecss --save-dev

Configuration

L'option "css" de la configuration doit être remplacée par les patrons du fichier de configuration principale. Pour l'option "content", seules les chaines de caractères sont gérées.

Les options "whitelist", "keyframes" et "fontFace" peuvent être utilisées.

Exemple

  • metalint.json
  • purgecss.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.css",
            "linters": "purgecss"
        }
    ]
}
          

{
    "content": "*.html"
}
