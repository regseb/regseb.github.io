Linters · JSONLint

Description

JSONLint est un outil pour analyser les fichiers contenant un objet JSON.

Autre linter JSON : JSON-Lint.

Installation

La commande suivante installe et ajoute JSONLint dans les dépendances de développement de votre projet npm. 


npm install jsonlint --save-dev

Configuration

Ce linter ne fournit aucune configuration.

Exemple

  • metalint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.json",
            "linters": { "jsonlint": null }
        }
    ]
}
