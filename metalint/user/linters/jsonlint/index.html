JSONLint

Description

JSONLint est un outil pour vérifier les fichiers contenant un objet JSON.

Installation

JSONLint doit-être installé avec une des deux commandes suivantes selon que Metalint ait été installé en global ou dans les dépendances de dévéloppement d'un paquet. 


npm install -g jsonlint
npm install --save-dev jsonlint

Configuration

Ce linter ne fournit aucune configuration.

Exemple

  • metalint.json

{
    "patterns": ["!node_modules/**", "**"],
    "checkers": [
        {
            "patterns": "**/*.json",
            "linters": { "jsonlint": null }
        }
    ]
}
