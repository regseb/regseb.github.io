JSCS

Description

JSCS est un outil pour vérifier le style d'un code source JavaScript.

JSCS est obsolète ! Il va être intégré dans le linter ESLint.

Configuration

Le fichier de configuration est identique à celui utilisé directement par JSCS. La configuration inline fonctionne aussi avec Metalint.

Toutes les règles peuvent être dans Metalint. Pour les options, la majorité sont utilisables directement dans le fichier de configuration de JSCS, sauf :

  • L'option "excludeFiles" qui permet d'exclure des fichiers. Pour remplacer cette option : utilisez les patrons dans le fichier de configuration metalint.json.
  • Grâce à l'option "fileExtensions", on peut filtrer les fichier à valider en fonction de leur extension. Il faut utiliser les patrons dans le fichier de configuration metalint.json.
  • Pour les options "extract", "maxErrors" et "fix" : il n'est pas possible de les utiliser dans Metalint.
  • L'option "verbose" doit être remplacée par l'option "verbose" du fichier de configuration metalint.json.
