CSSLint

Description

CSSLint est un outil pour vérifier les feuilles de style (CSS).

Configuration

Toutes les règles peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • csslint.json

{
    "patterns": ["!node_modules/**", "**"],
    "checkers": [
        {
            "patterns": "**/*.css",
            "linters": "csslint"
        }
    ]
}
          

{
    "empty-rules": true,
    "floats": true,
    "ids": true,
    "important": true,
    "known-properties": true,
    "order-alphabetical": true,
    "zero-units": true
}
