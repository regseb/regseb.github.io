Linters · CSSLint

Description

CSSLint est un outil pour analyser les feuilles de style (CSS).

Autres linters CSS (SCSS, Sass, Less) : doiuse, lesshint, Purgecss et stylelint.

Installation

La commande suivante installe et ajoute CSSLint dans les dépendances de développement de votre projet npm. 


npm install csslint --save-dev

Configuration

Toutes les règles peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • csslint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.css",
            "linters": "csslint"
        }
    ]
}
          

{
    "empty-rules": true,
    "floats": true,
    "ids": true,
    "important": true,
    "known-properties": true,
    "order-alphabetical": true,
    "zero-units": true
}
