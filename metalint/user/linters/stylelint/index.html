Linters · stylelint

Description

stylelint est un outil pour analyser les feuilles de style (CSS).

Autres linters CSS (SCSS, Sass, Less) : CSSLint, doiuse, lesshint et Purgecss.

Installation

La commande suivante installe et ajoute stylelint dans les dépendances de développement de votre projet npm. 


npm install stylelint --save-dev

Configuration

Toutes les règles peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • stylelint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.css",
            "linters": "stylelint"
        }
    ]
}
          

{
    "rules": {
        "color-hex-case": "lower",
        "color-hex-length": null,
        "color-no-invalid-hex": true,

        "font-family-name-quotes": "always-where-recommended",

        "font-weight-notation": ["named-where-possible", {
            "ignore": ["relative"]
        }]
    }
}
