stylelint

Description

stylelint est un outil pour vérifier les feuilles de style (CSS).

Installation

stylelint doit-être installé avec une des deux commandes suivantes selon que Metalint ait été installé en global ou dans les dépendances de dévéloppement d'un paquet. 


npm install -g stylelint
npm install --save-dev stylelint

Configuration

Toutes les règles peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • stylelint.json

{
    "patterns": ["!node_modules/**", "**"],
    "checkers": [
        {
            "patterns": "**/*.css",
            "linters": "stylelint"
        }
    ]
}
          

{
    "rules": {
        "color-hex-case": "lower",
        "color-hex-length": null,
        "color-no-invalid-hex": true,

        "font-family-name-quotes": "always-where-recommended",

        "font-weight-notation": ["named-where-possible", {
            "ignore": ["relative"]
        }]
    }
}
