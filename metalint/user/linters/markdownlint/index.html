MarkdownLint

Description

MarkdownLint est un outil pour analyser le code source des fichiers Markdown.

Installation

Markdownlint doit-être installé avec une des deux commandes suivantes selon que Metalint ait été installé en global ou dans les dépendances de dévéloppement d'un paquet. 


npm install -g markdownlint
npm install --save-dev markdownlint

Configuration

Le fichier de configuration doit contenir un objet JSON dont les clés sont les codes des règles et les valeurs : les paramètres de la règles.

Toutes les règles peuvent être dans Metalint.

Exemple

  • metalint.json
  • markdownlint.json

{
    "patterns": ["!node_modules/**", "**"],
    "checkers": [
        {
            "patterns": "**/*.md",
            "linters": "markdownlint"
        }
    ]
}
          

{
    "MD003": { "style": "atx" },
    "MD004": { "style": "dash" },
    "MD029": { "style": "ordered" },
    "MD035": { "style": "---" }
}
