Linters · MarkdownLint

Description

MarkdownLint est un outil pour analyser le code source des fichiers Markdown.

Installation

La commande suivante installe et ajoute Markdownlint dans les dépendances de développement de votre projet npm. 


npm install markdownlint --save-dev

Configuration

Le fichier de configuration doit contenir un objet JSON dont les clés sont les codes des règles. Et les valeurs : les paramètres de la règle.

Toutes les règles peuvent être dans Metalint.

Exemple

  • metalint.json
  • markdownlint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.md",
            "linters": "markdownlint"
        }
    ]
}
          

{
    "heading-style": { "style": "atx" },
    "ul-style": { "style": "dash" },
    "ol-prefix": { "style": "ordered" },
    "hr-style": { "style": "---" }
}
