Linters · lesshint

Description

lesshint est un outil pour analyser les fichiers Less.

Autres linters CSS (SCSS, Sass, Less) : CSSLint, doiuse, Purgecss et stylelint.

Installation

La commande suivante installe et ajoute lesshint dans les dépendances de développement de votre projet npm. 


npm install lesshint --save-dev

Configuration

Toutes les règles peuvent être utilisées dans Metalint. Sauf les deux options spéciales "fileExtensions" et "excludedFiles" qui doivent être remplacées par les patrons du fichier de configuration principale.

Exemple

  • metalint.json
  • lesshint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": ["!vendor.less", "*.less", "*.css"],
            "linters": "lesshint"
        }
    ]
}
          

{
    "spaceAfterPropertyColon": {
        "enabled": true,
        "style": "one_space"
    },

    "emptyRule": true,

    "importantRule": false
}
