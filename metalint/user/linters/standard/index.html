JavaScript Standard Style

Description

JavaScript Standard Style est un analyseur syntaxique pour les codes JavaScript. Son but est de proposer un linter simple d'utilisation et tenter de fournir un standard.

Configuration

Ce linter n'a aucune configuration. Les règles sont fixes (il n'est pas possible d'en désactiver).

Exemple

  • metalint.json

{
    "patterns": ["!node_modules/**", "**"],
    "checkers": [
        {
            "patterns": ["!*/*.min.js", "**/*.js"],
            "linters": "standard"
        }
    ]
}
