Linters · YAML Lint

Description

YAML Lint est un analyseur syntaxique pour les fichiers YAML.

Installation

La commande suivante installe et ajoute YAML Lint dans les dépendances de développement de votre projet npm. 


npm install yaml-lint --save-dev

Configuration

YAML Lint possède deux options :

  • "schema" qui peut être utilisée dans Metalint ;
  • "ignore" : il faut utiliser les patrons du fichier de configuration principale pour exclure des fichiers.

Exemple

  • metalint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.json",
            "linters": { "yaml-lint": { "schema": "CORE_SCHEMA" } }
        }
    ]
}
