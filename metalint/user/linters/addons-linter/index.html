Add-ons Linter

Description

Add-ons Linter est un validateur pour les add-ons (XPI) du navigateur Firefox.

Installation

Add-ons Linter doit-être installé avec une des deux commandes suivantes selon que Metalint ait été installé en global ou dans les dépendances de dévéloppement d'un paquet. 


npm install -g addons-liner
npm install --save-dev addons-linter

Configuration

Les règles ne sont pas configurables.

Exemple

  • metalint.json

{
    "patterns": ["!node_modules/**", "**"],
    "checkers": [
        {
            "patterns": "**/*.xpi",
            "linters": { "addons-linter": null }
        }
    ]
}

Autres linters pour les add-ons Firefox

Add-ons Linter est le seul linter géré par Metalint pour analyser les add-ons Firefox.
