Linters · Add-ons Linter

Description

Add-ons Linter est un validateur pour les add-ons du navigateur Firefox (WebExtension).

Installation

La commande suivante installe et ajoute Add-ons Linter dans les dépendances de développement de votre projet npm. 


npm install addons-linter --save-dev

Configuration

Les règles ne sont pas configurables.

Exemple

  • metalint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.xpi",
            "linters": { "addons-linter": null }
        }
    ]
}
