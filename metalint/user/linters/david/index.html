Linters · David DM

Description

david est un utilitaire qui signale les dépendances obsolètes dans votre package.json. C'est l'exécutable qui se cache derrière le site david-dm.org.

Installation

La commande suivante installe et ajoute david dans les dépendances de développement de votre projet npm. 


npm install david --save-dev

Configuration

La configuration est un objet JSON dont les propriétés peuvent-être : "dev", "optional" et "peer". Et les valeurs, un booléen (true ou false) pour activer / désactiver la vérification des différentes dépendances. Les dépendances principales sont toujours analysées. Par défaut, il faut fournir un objet vide : {}.

Exemple

  • metalint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "/package.json",
            "linters": { "david": { "dev": true } }
        }
    ]
}

Cet exemple vérifie les dépendances principales et celles de développements.
