Linters · doiuse

Description

doiuse est un outil pour analyser les feuilles de style (CSS).

Autres linters CSS (SCSS, Sass, Less) : CSSLint, lesshint, Purgecss et stylelint.

Installation

La commande suivante installe et ajoute doiuse dans les dépendances de développement de votre projet npm. 


npm install doiuse --save-dev

Configuration

Les règles browsers et ignore peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • doiuse.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.css",
            "linters": "doiuse"
        }
    ]
}
          

{
    "browsers": ["ie >= 8", "> 1%"],
    "ignore": ["rem"]
}
