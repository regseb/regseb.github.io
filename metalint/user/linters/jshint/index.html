JSHint

Description

JSHint est un analyseur syntaxique pour les codes JavaScript.

Configuration

Le fichier de configuration est identique à celui utilisé directement par JSHint. La configuration inline fonctionne aussi avec Metalint.

Toutes les options peuvent être utilisées dans Metalint. Sauf les deux options spéciales :

  • L'option "extends" permet d'hériter d'un autre fichier de configuration. Pour remplacer cette option : passer plusieurs fichiers de configuration au linter dans le fichier metalint.json.
  • Grâce à l'option "overrides", on peut spécifier des règles particulières pour certains fichiers. Dans Metalint, cette gestion de configurations différentes selon les fichiers est faite avec les checkers du fichier metalint.json.

Pour le fichier .jshintignore : utiliser les patrons du fichiers de configuration principale pour exclure des fichiers.
