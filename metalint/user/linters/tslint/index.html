Linters · TSLint

Description

TSLint est un analyseur syntaxique pour les codes TypeScript.

Installation

La commande suivante installe et ajoute TSLint (et le paquet typescript) dans les dépendances de développement de votre projet npm. 


npm install tslint typescript --save-dev

Configuration

Toutes les règles peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • tslint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.ts",
            "linters": "tslint"
        }
    ]
}
          

{
    "rules": {
        "max-line-length": {
            "options": [120]
        },
        "new-parens": true,
        "no-arg": true,
        "no-bitwise": true,
        "no-conditional-assignment": true,
        "no-consecutive-blank-lines": false,
        "no-console": {
            "severity": "warning",
            "options": ["debug", "info", "log", "time", "timeEnd", "trace"]
        }
    }
}
