Linters · CoffeeLint

Description

CoffeeLint est un analyseur syntaxique pour les codes CoffeeScript.

Installation

La commande suivante installe et ajoute CoffeeLint dans les dépendances de développement de votre projet npm. 


npm install coffeelint --save-dev

Configuration

Le fichier de configuration est identique à celui utilisé directement par CoffeeLint.

Exemple

  • metalint.json
  • coffeelint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.coffee",
            "linters": "coffeelint"
        }
    ]
}
          

{
    "indentation": {
        "level": "error",
        "value": 4
    },
    "line_endings": {
        "value": "unix",
        "level": "error"
    }
}
