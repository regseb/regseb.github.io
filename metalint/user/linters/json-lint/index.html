Linters · JSON-Lint

Description

JSON-Lint est un outil pour analyser les fichiers contenant un objet JSON. Les commentaires peuvent être autorisés avec ce linter.

Autre linter JSON : JSONLint.

Installation

La commande suivante installe et ajoute JSON-Lint dans les dépendances de développement de votre projet npm. 


npm install json-lint --save-dev

Configuration

JSON-Lint possède une seule option (pour les commentaires) et elle est utilisable dans Metalint.

Exemple

  • metalint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": "*.json",
            "linters": { "json-lint": { "comments": false } }
        }
    ]
}
