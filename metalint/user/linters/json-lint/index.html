JSON-Lint

Description

JSON-Lint est un outil pour vérifier les fichiers contenant un objet JSON. Les commentaires peuvent être autorisés avec ce linter.

Configuration

JSON-Lint possède une seule option (pour les commentaires) et elle est utilisable dans Metalint.
