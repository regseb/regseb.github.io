Linters · Flow

Description

Flow est un outil pour analyser le typage des variables JavaScript.

Autres linters JavaScript : ESLint, JSCS, JSHint et JavaScript Standard Style.

Installation

La commande suivante installe et ajoute flow-bin (l'encapsulation binaire de Flow) dans les dépendances de développement de votre projet npm. 


npm install flow-bin --save-dev

Configuration

Il n'y a pas de configuration pour ce linter.

Il faut ajouter un fichier vide, portant le nom .flowconfig, à la racine de votre projet.

Exemple

  • metalint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": ["!*.min.js", "*.js"],
            "linters": { "flow-bin": null }
        }
    ]
}
