Flow

Description

Flow est un outil pour analyser le typage des variables JavaScript.

Installation

flow-bin (l'encapsulation binaire de Flow) doit-être installé avec une des deux commandes suivantes selon que Metalint ait été installé en global ou dans les dépendances de dévéloppement d'un paquet. 


npm install -g flow-bin
npm install --save-dev flow-bin

Configuration

Il n'y a pas de configuration pour ce linter.

Il faut ajouter un fichier vide, portant le nom .flowconfig, à la racine de votre projet.

Exemple

  • metalint.json

{
    "patterns": ["!node_modules/**", "**"],
    "checkers": [
        {
            "patterns": ["!**/*.min.js", "**/*.js"],
            "linters": { "flow-bin": null }
        }
    ]
}
