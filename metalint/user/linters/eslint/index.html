ESLint

Description

ESLint est un analyseur syntaxique pour les codes JavaScript.

Configuration

Le fichier de configuration est identique à celui utilisé directement par ESLint.

Toutes les règles peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • eslint.json

{
    "patterns": ["**", "!node_modules/**"],
    "checkers": [
        {
            "patterns": ["**/*.js", "!**/*.min.js"],
            "linters": "eslint"
        }
    ]
}
          

{
    "ecmaFeatures": {
        "blockBindings": true,
        "forOf": true
    },
    "rules": {
        "valid-jsdoc": [2, {
            "prefer": {
                "returns": "return",
                "class":   "constructor"
            },
            "requireReturn": false
        }]
    }
}
