ESLint

Description

ESLint est un analyseur syntaxique pour les codes JavaScript.

Installation

ESLint doit-être installé avec une des deux commandes suivantes selon que Metalint ait été installé en global ou dans les dépendances de dévéloppement d'un paquet. 


npm install -g eslint
npm install --save-dev eslint

Configuration

Le fichier de configuration est identique à celui utilisé directement par ESLint.

Toutes les règles peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • eslint.json

{
    "patterns": ["!node_modules/**", "**"],
    "checkers": [
        {
            "patterns": ["!**/*.min.js", "**/*.js"],
            "linters": "eslint"
        }
    ]
}
          

{
    "parserOptions": {
        "ecmaFeatures": {
            "blockBindings": true,
            "forOf": true
        }
    },

    "globals": {
        "Promise": true,
    },

    "rules": {
        "comma-dangle": 2,
        "no-undef": 2,
        "no-warning-comments": 1,
        "prefer-const": 2,
        "radix": 2,
        "strict": [2, "global"],
    }
}
