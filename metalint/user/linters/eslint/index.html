Linters · ESLint

Description

ESLint est un analyseur syntaxique pour les codes JavaScript.

Autres linters JavaScript : Flow, JSCS, JSHint et JavaScript Standard Style.

Installation

La commande suivante installe et ajoute ESLint dans les dépendances de développement de votre projet npm. 


npm install eslint --save-dev

Configuration

Le fichier de configuration est identique à celui utilisé directement par ESLint.

Toutes les règles peuvent être utilisées dans Metalint.

Exemple

  • metalint.json
  • eslint.json

{
    "patterns": ["!/node_modules/", "**"],
    "checkers": [
        {
            "patterns": ["!*.min.js", "*.js"],
            "linters": "eslint"
        }
    ]
}
          

{
    "parserOptions": {
        "ecmaFeatures": {
            "blockBindings": true,
            "forOf": true
        }
    },

    "globals": {
        "Promise": true,
    },

    "rules": {
        "comma-dangle": 2,
        "no-undef": 2,
        "no-warning-comments": 1,
        "prefer-const": 2,
        "radix": 2,
        "strict": [2, "global"],
    }
}
