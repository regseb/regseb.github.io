Installation

Prérequis

Node.js v6 (ou plus) est nécessaire pour utiliser Metalint. Node.js est un outil permettant d'exécuter du code JavaScript directement sur la machine, sans utiliser de navigateur Internet.

Téléchargement

Metalint est hébergé dans le gestionnaire de paquets npm. Node.js fournit l'utilitaire npm qui permet de télécharger un paquet. En exécutant la commande ci-dessous, Metalint sera téléchargé et l'éxécutable metalint sera ajouté à votre PATH. 


npm install -g metalint

Il faut ensuite installer en global les linters que vous comptez utiliser.

Intégration

Si vous utilisez npm dans votre projet, vous pouvez ajouter Metalint dans les dépendances de développement : 


npm install --save-dev metalint

Il faut ensuite ajouter aussi les linters dans les dépendances de développement.

Puis ajoutez la propriété "lint" dans les "scripts" du fichier package.json : 


{
    "scripts": {
        "lint": "metalint"
    }
}

Metalint sera utilisable avec la commande : npm run lint
