Installation

Prérequis

Node.js 4.3 (ou plus) est nécessaire pour utiliser Metalint. Node.js est un outil permettant d'exécuter du code JavaScript directement sur la machine, sans utiliser de navigateur Internet.

Téléchargement

Metalint est hébergé dans le gestionnaire de paquets npm. Node.js fournit l'utilitaire npm qui permet de télécharger un paquet. En exécutant la commande ci-dessous, Metalint sera téléchargé et l'éxécutable metalint sera ajouté à votre PATH. 


npm install -g metalint

Intégration

Si vous utilisez npm dans votre projet, vous pouvez y intégrer le lancement de Metalint. Pour cela : insérez Metalint dans les dépendances de développement du fichier package.json. Puis ajoutez la propriété "lint" dans les "scripts" : 


{
    "scripts": {
        "lint": "metalint"
    },
    "devDependencies": {
        "metalint": "^0.3.1"
    }
}

Après avoir installé les dépendances de votre projet : npm install, vous pourrez lancer Metalint avec la commande suivante : 


npm run lint

Mise à jour

Pour mettre à jour Metalint, il vous suffit d'entrer la ligne de commande suivante : 


npm update -g metalint

La numérotation de version respecte les spécifications de la gestion sémantique de version 2.0.0 (SemVer). Voici un extrait des spécifications :

Étant donné un numéro de version MAJEUR.MINEUR.CORRECTIF, il faut incrémenter :

  1. le numéro de version MAJEUR quand il y a des changements rétro-incompatibles,
  2. le numéro de version MINEUR quand il y a des changements rétro-compatibles,
  3. le numéro de version de CORRECTIF quand il y a des corrections d’anomalies rétro-compatibles

Des libellés supplémentaires peuvent être ajoutés pour les versions de pré-livraison et pour des méta-données de construction sous forme d'extension du format MAJEURE.MINEURE.CORRECTIF.
