Installation

Prérequis

Node.js 10+ est nécessaire pour utiliser Metalint.

Intégration

Metalint est hébergé dans le gestionnaire de paquets npm. En exécutant la commande ci-dessous, Metalint sera téléchargé et ajouté dans les dépendances de développement de votre projet. 


npm install metalint --save-dev

Pour simplifier l'utilisation de Metalint (avec la commande npm run lint), vous pouvez ajouter la propriété "lint" dans les "scripts" du fichier package.json : 


{
    "scripts": {
        "lint": "metalint"
    }
}

Initialisation

Pour créer un fichier de configuration, placez-vous à la racine de votre projet et exécutez la commande suivante : 


npx metalint init

Le répertoire .metalint sera ajouté dans votre projet. Il contiendra le fichier metalint.json qui listera tous les linters présents dans vos dépendances de développement et tentera de lier Metalint avec les fichiers de configuration des linters.

Metalint est maintenant utilisable avec la commande : npx metalint (ou npm run lint).
