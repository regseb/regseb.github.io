Rapporteurs

Introduction

Metalint peut afficher les résultats dans différents formats. Ce choix se fait avec la propriété "reporter" du fichier de configuration metalint.json. Ou avec l'option --reporter de la ligne de commande.

Les rapporteurs disponibles sont :

  • checkstyle : le résultat est écrit dans un document XML respectant les conventions des rapports de Checkstyle ;
  • console (par défaut) : un format adapté pour l'affichage des résultats dans une console et l'analyse par un humain.
  • csv : les données sont affichées dans un format CSV où chaque ligne représente une alerte ;
  • json : les résultats brutes de Metalint au format JSON.
  • none : aucun résultat n'est affiché (la valeur de retour de l'exécutable est quand même renseignée).
  • unix : un format compacte affichant l'essentiel des informations des alertes.

Si vous souhaitez créer un autre format, consultez la page Rapporteur du guide développeur.
