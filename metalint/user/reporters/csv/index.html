csv (Rapporteur)

Description

Ce rapporteur retourne les données au format CSV. La première ligne contient le nom des colonnes et chaque ligne suivante correspond à une alerte. Les colonnes sont :

  • file : le nom du fichier ;
  • line : le numéro de la ligne ;
  • column : le numérode la colonne ;
  • message : le message décrivant le problème ;
  • linter (si la verbosité est supérieure ou égale 1) : le linter ayant trouvé le problème ;
  • rule (si la verbosité est supérieure ou égale à 2) : la règle qui n'est pas respectée.

Les colonnes sont séparées par une virgule et les lignes par les caractères CRLF (\r\n). Les valeurs des colonnes file, message et rule sont enrobées par des guillemets.

Exemple

Voici un rapport (avec la verbosité à 2) où deux alertes sont affichés pour le même fichier. 


file,line,column,message,linter,rule
"script.js",255,36,"Don't make functions within a loop.",jshint,"W083"
"script.js",278,3,"Comments cannot contain the following keywords: todo, fixme",jscs,
