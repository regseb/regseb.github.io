console (Rapporteur)

Description

Avec ce rapporteur, un extrait du code source est affiché pour chaque alertes remontées.

La verbosité peut varier en 0 et 4:

  • 1 : ajouter le nom linter ayant détecté le problème ;
  • 2 : ajouter le code de la règle qui n'est pas respecté ;
  • 3 : afficher les fichiers qui ont été vérifiés, mais qui n'ont pas de problème ;
  • 4 : afficher les fichiers respectant les patrons globaux, mais qu'ils n'ont été vérifiés par aucun linter.
