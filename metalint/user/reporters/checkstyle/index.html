checkstyle (Rapporteur)

Description

Ce rapporteur retourne un document XML ayant la structure des rapports de l'outil Checkstyle.

En passant le niveau de verbosité à 1 : le document XML est indenté.

Exemple

Voici un rapport (avec la verbosité à 1) où deux fichiers sont vérifiés : global.js qui n'a aucune erreur et script.js qui contient deux erreurs (une remontée par JSHint et l'autre par JSCS). 


<?xml version="1.0" encoding="UTF-8"?>
<checkstyle version="6.8">
  <file name="global.js">
  </file>
  <file name="script.js">
    <error line="255" column="36" severity="error" message="Don&apos;t make functions within a loop." source="jshint.W083" />
    <error line="278" column="3" severity="error" message="Comments cannot contain the following keywords: todo, fixme" source="jscs" />
  </file>
</checkstyle>
